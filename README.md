# Tally CLI

Command-line interface for interacting with the Tally Protocol on Solana.

## Installation

```bash
cargo install --git https://github.com/Tally-Pay/tally-cli
```

## Building from Source

```bash
git clone https://github.com/Tally-Pay/tally-cli
cd tally-cli
cargo build --release
```

## Usage

The CLI provides commands for managing merchants, plans, and subscriptions on the Tally Protocol.

```bash
tally-cli --help
```

## Dependencies

This CLI depends on the [tally-sdk](https://github.com/Tally-Pay/tally-protocol) for interacting with the Tally Protocol program.
